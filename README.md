# README

## INSTALL

```
npm install
bower install
```

## Development

```
ln -s config_example.js public/config.js
```


### Preview
```
npm start
```
or
```
grunt serve
```
### Build
```
grunt build
```
*Preview Build*
```
grunt serve-build
```
#### Build and Release Version
```
grunt release
```

## Deployment
```
git clone -b build https://github.com/monitormap/webclient.git
```
